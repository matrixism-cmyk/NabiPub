# 나비드라이브 (NabiDrive)

Windows용 SFTP/FTP 가상 드라이브. 원격 서버를 로컬 드라이브처럼 사용합니다.

## 주요 기능
- **Dokan 가상 드라이브** — 커스텀 볼륨명, 원격 서버가 로컬 드라이브로 표시
- **SFTP / FTP 지원** — SSH 기반 보안 연결 + FTP 호환
- **원격 디렉토리 브라우저** — 연결 설정 시 경로 탐색 선택
- **자동 업데이트** — 앱 내에서 새 버전 감지 + 다운로드 + 설치
- **디스크 캐시** — LRU 캐시로 반복 접근 속도 향상
- **Windows 통합** — 트레이 아이콘, 시작 프로그램, 풍선 알림

## 설치
[Releases](https://github.com/matrixism-cmyk/NabiDrivePub/releases) 페이지에서 최신 `nabidrive_setup_vXXX.exe` 다운로드 후 실행

## 시스템 요구사항
- Windows 10/11 (64비트)
- Dokan 드라이버 (인스톨러에서 자동 설치)
